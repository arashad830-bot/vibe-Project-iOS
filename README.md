# Vibe Project - iOS

iOS native app built with **Swift** and **SwiftUI**.

## Project Structure

```
VibeProject/
├── App/
│   ├── Views/
│   ├── ViewModels/
│   ├── Models/
│   └── Services/
├── Resources/
│   ├── Assets/
│   ├── Localizable.strings
│   └── Info.plist
└── Tests/
```

## Tech Stack

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Architecture**: MVVM
- **Min iOS**: 14.0

## Getting Started

1. Install Xcode 14+
2. Open VibeProject.xcodeproj
3. Select target device
4. Run (⌘R)
